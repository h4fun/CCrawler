## CCrawler

一个分布式自动化爬虫



#### 1. 目标

- 通过web界面添加规则，后台自动化爬取目标网页
- 可定义存储类型-> (redis or mysql or hbase or hdfs or mq)
- 可监控爬取状态，提供日志监控，可配置报警规则
- 具有灵活的高可用性
